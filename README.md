## How to install?
- git (github)


Install from Github
```
$ git clone https://github.com/happysources/unittest-parser.git
$ cd unittest-parser
$ pip install -r requirements.txt
```

## Examples:

```
$ python3 unittesto_parser.py test/example/test_multi.xml > /tmp/test.html
```
